The Complete Javascript Course: From Zero to Expert

This is a Javascript course on Udemy by Jonas Schmedtmann, It includes the following: • Built 5 beautiful real-world projects! In these projects, I learned how to plan and architect myapplications using flowcharts and common JavaScript patterns.

• Learn how to use some tools including: NPM, Parcel, Babel, and ES6 modules.
